Exercicios com estilos CSS

Atualmente existem 3 formas de aplicar estilos em CSS ao HTML, são elas INTERNOS, EXTERNOS e INLINE, veremos cada um deles aqui na pratica

Inline

O primeiro estilo é inline, que é a forma mais facil porem que mais nos prejudicara na hora de modificar as tags, pois são estilos aplicados diretamente na tag, que é o estilo que prevalece.

INTERNA

As CSS aplicadas no estilos internos é melhor que a primeira pois nos possibilita alterar o corpo da pagina, mas isso só ficaria bom se usassemos até no maximo 2 ou 3 paginas, pois se não ficaria muito complicado alterar pagina a pagina.

EXTERNA

Para isso usremos a tag link que nos possibilitara criar 3 links, um que sera nosso index, o outro que sera o link de exemplo da pagina02.html e o terceiro sera nosso link de CSS para modificar os estilos de todas as paginas pode ser elas 2 ou 30 vamos mudar todas da melhor forma com esse estilo css externo.(Essa e a melhor e mais usada forma de colocar CSS)
